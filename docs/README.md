# PersonaPlex Documentation

This directory contains detailed documentation for the PersonaPlex codebase.

## Available Documentation

### [LM_ARCHITECTURE.md](./LM_ARCHITECTURE.md)
Comprehensive explanation of the language model architecture implemented in `moshi/moshi/models/lm.py`. This document covers:

- **Overview**: Purpose and role of the language model in PersonaPlex
- **Key Components**: Detailed breakdown of classes, methods, and data structures
- **Architecture Details**: Two-stage transformer design, delay patterns, and streaming mechanisms
- **Usage Examples**: Training and inference flows
- **Performance Considerations**: Memory efficiency and latency optimization

Perfect for developers who want to understand how PersonaPlex processes text and audio tokens for real-time conversational speech.
